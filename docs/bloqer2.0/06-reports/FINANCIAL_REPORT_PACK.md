# Paquete de reportes financieros

## Contenido estándar mensual
1. Posición de tesorería y cashflow real (**caja ejecutada** únicamente).
2. Proyección de caja 90 días (**liquidez** vía AR/AP; no confundir con costo comprometido).
3. Aging AR y AP.
4. Rentabilidad por proyecto (bruta y neta si permiso) — **declarar capa de costo**: devengado, pagado o exposición esperada ([`COST_FORMULAS.md`](../04-formulas/COST_FORMULAS.md), [BR-COS-001]).
5. Impuestos y retenciones del período.
6. Conciliación bancaria por cuenta.

**Convención transversal:** costo (`committed` / `accrued` / `paid` / `expected_cost_exposure`) ≠ cashflow real ≠ proyección — ver §2 en [`COST_FORMULAS.md`](../04-formulas/COST_FORMULAS.md).

## Audiencia
Director financiero, contador externo (export).
